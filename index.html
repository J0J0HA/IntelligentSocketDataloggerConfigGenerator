ISDL Config Editor

This is a config editor and file generator for Technik-Tueftler/IntelligentSocketDatalogger and offers a graphical way to generate/edit the config.json, devices.json and device_plugin.py Files.

Select one of the following Options:

  • Edit Config (config.json and devices.json)
  • Edit Plugins (device_plugin.py)
